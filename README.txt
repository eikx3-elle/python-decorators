# Python Decorators

A collection of production-ready decorators built from scratch.

## Decorators included

- `@timer` — measures execution time of any function
- `@retry` — automatically retries a function on failure, configurable attempts
- `@log` — logs function calls, arguments, and return values

## Concepts demonstrated

- Functions as first-class objects
- Closures and higher order functions
- *args and **kwargs for universal argument passing
- functools.wraps for preserving function identity
- Decorator factories with configurable arguments

## Run it

python decorators.py
